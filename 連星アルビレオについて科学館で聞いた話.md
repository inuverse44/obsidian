# 要約
- 白鳥座の$\beta$星で、２重連星。

# 詳細メモ
- アルビレオ (Albireo)は白鳥座の連星である。
- $\beta$星である（白鳥座で2番目に明るい星）
- 赤色と青色の星の２重星である。
	- 赤よりの星の方が青色の星よりも、星形成されて時間が経過している
	- 詳しくはヘルツシュプルング・ラッセル図参照
